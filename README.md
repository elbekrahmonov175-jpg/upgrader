# SkinUp — демо-клон апгрейдера скинов (учебный проект)

Полностью виртуальный клон сайтов типа upgrader.pro:
- своя регистрация/вход (**без Steam**)
- виртуальный баланс и виртуальные "скины" (**без реальных выводов**)
- механика апгрейда: ставишь предмет или баланс → выбираешь множитель →
  крутится полоса шанса → выигрыш/проигрыш
- история апгрейдов

Все названия скинов — выдуманные, картинки — CSS-градиенты по редкости
(чтобы не тащить в проект чужой контент из настоящей игры).

## Стек

- Flask + Flask-Login + Flask-SQLAlchemy
- PostgreSQL (на Railway) / SQLite (локально, автоматически)
- Ванильный JS для анимации полосы шанса

## Локальный запуск

```bash
pip install -r requirements.txt
python app.py
```

Открой http://localhost:5000/register — без переменных окружения база
будет локальный файл `local.db` (SQLite), этого достаточно для теста.

## Деплой на Railway (как обычно)

1. Залей проект на GitHub.
2. В Railway: New Project → Deploy from GitHub repo → выбери репозиторий.
3. Добавь плагин **PostgreSQL** в этом же проекте Railway — он сам создаст
   переменную `DATABASE_URL` и подключит её к сервису.
4. В Variables сервиса добавь:
   - `SECRET_KEY` — любая случайная строка (например, сгенерируй через
     `python -c "import secrets; print(secrets.token_hex(32))"`)
5. Railway сам подхватит `Procfile` и запустит `gunicorn app:app`.
6. При первом запуске таблицы создаются автоматически (`db.create_all()`),
   и база наполняется демо-скинами (`seed_skins.py`).

## Структура проекта

```
app.py              — роуты и логика апгрейда
models.py           — модели User / Skin / InventoryItem / UpgradeHistory
config.py           — настройки (RTP, стартовый баланс, БД)
seed_skins.py        — список демо-скинов
templates/           — HTML-шаблоны (Jinja2)
static/css/style.css — стили
static/js/upgrade.js — логика и анимация апгрейда
```

## Что можно докрутить дальше

- Своя админка для добавления/редактирования скинов через веб-интерфейс.
- Реальные картинки скинов — просто положи файлы в `static/skins/` и
  добавь поле `image_url` в модель `Skin`.
- Rate-limit на `/api/upgrade`, чтобы никто не спамил запросы напрямую
  мимо интерфейса.
- Таблица лидеров (топ по балансу/самым дорогим выбитым скинам).

## Важно

Это учебный/демо-проект с виртуальной валютой. Если планируешь
показывать его публично, стоит явно обозначить на сайте, что это
демо без реальных денег/выводов, чтобы не создавать впечатление
настоящей гэмблинг-платформы.
